Về TechMart

Đây là trang demo TMDT tối giản: có navbar, footer, trang chủ hiển thị sản phẩm (mock), và trang 404. Bạn có thể dùng nó làm “template” cho bài tập/đồ án rồi mở rộng thêm cart/checkout.

Điểm nhấn: CSS + JS nhúng trực tiếp trong HTML (đúng yêu cầu), dễ copy-paste chạy nhanh.

Bạn có thể mở rộng

• Trang danh mục / chi tiết sản phẩm
• Trang giỏ hàng / thanh toán
• Login/Signup, phân quyền
• Gọi API thật và phân trang

Tính năng demo

• Navbar + logo quay về Home
• Footer liên kết nhanh
• Điều hướng 404 và nút quay về Home
